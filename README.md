# Проект "Crossword"
**Выполненная работа:**
* Разработка генератора кроссвордов (JS)

**Запуск:**
* `node script.js`

**Параметры:**

* MAX_WORDS: максимальная длина реально используемого словаря. Для демонстрации достаточно 15.
* field.txt - формат поля. Звездочки (*) - ячейки для заполнения. Прочие символы (.) - вычеркнутые ячейки.
* src_sorted.txt - общий словарь.

Результат выводится в консоль.
