# Проект "Crossword"
**Выполненная работа:**
* Разработка генератора кроссвордов (JS)

**Запуск:**
* `node script.js`

**Параметры:**

* MAX_WORDS: максимальная длина реально используемого словаря. Для демонстрации достаточно 23.
* field.txt - формат поля. Символы подчеркивания (_) - ячейки для заполнения. Прочие символы (*) - вычеркнутые ячейки.
* words.txt - общий словарь.

Результат выводится в консоль.
